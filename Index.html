FULL-STACK WEB DEVELOPMENT USING
THE MERN STACK


What is "MERN"?

The 'MERN Stack' Consists of four libraries of the JavaScript i.e,

  • Mongo DB
  • Express.js
  • React.js
  • Node.js


As Full-Stack Web development deals with Front-end, Back-end, as well as Database layer.
In 'front-end layer' the developer work with the interface that the user can see while accessing,
a web site/application,The engineers who work in 'front-end layer' are known as Front-end
Engineers/Developer.



Now coming to the 'back-end layer' the developer works with the logics and functionality of the
web application. It is also known as the 'Brain' of your web application. The engineers who work in
'back-end layer' are known as Back-end Engineers/Developer.



The third layer was the 'database layer' in this the developer works with the data and information
that is stored by the website. The engineers has to work with Mongo DB, Node.js, SQL, etc.
They are known as the Database Engineers.

What is prerequisite ?

One should have familiar with the 'HTML & CSS', and also know the uses of different tags as per the
their function and uses. Also the knowledges of the various 'Attributes & Styles' used for the styling
and formatting the content of the HTML elements.
And for the 'Working & Functioning' of the webpage the knowledge of the JavaScript is also required,
as JavaScript makes the functioning of the web pages/applications done.
Also, how to link different files to each other as to proper Functioning and working of the web pages/applications.
